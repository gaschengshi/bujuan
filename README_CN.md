# Bujuan（P人工作台）

一个面向「容易分心但想持续推进」场景的 Obsidian 插件。  
它把“今天要做什么、下一步做什么、做完了什么”集中在一个独立工作台里，尽量降低启动成本。
注意！！！请事先安装好dataview插件，否则无法使用。

## 核心功能

- 独立工作台视图：卡片化展示今日目标与当前步骤。
- AI 下一步建议：一键生成 3 条候选步骤，点击即可选中。
- 当前进行管理：支持手动输入、确认选中、一步完结。
- 历史记录：查看已完成步骤，展示完成时间。
- 灵感笔记联动：记录灵感后，历史中的 `[[笔记]]` 可点击跳转。
- 目标状态管理：支持 `active / dormant / completed / archived`。
- 休眠机制：目标可休眠并在重新选择时自动唤醒。
- 重新选择今天任务：支持搜索、过滤，快速换当天聚焦目标。
- 随机抽目标：命令或丝带一键“随便选一个”。

## AI 支持

- 使用 OpenAI 兼容接口。
- 设置项可配置：`API Key`、`Base URL`、`Model`、Prompt 模板。
- 自动补全请求地址到 `/v1/chat/completions`。

## 如何使用

- 左侧丝带图标：
- `leaf`：打开 P人工作台
- `dice`：随机抽一个目标
- 命令面板（`Ctrl/Cmd + P`）：
- `打开P人工作台`
- `P人工作台：随便选一个`
- `新建目标`
- `重新选择`

## 快速上手

1. 在设置里先填好 AI 配置。
2. 通过“添加目标”创建/加入今天任务。
3. 在卡片中点击“AI建议”或直接手动输入当前步骤。
4. 点击“选中”确认后，用“下一步”完结当前步骤。
5. 需要补充想法时点“记录”，自动创建并打开灵感笔记。

## 适用人群

- 想把大目标拆成可执行小步的人。
- 需要“低摩擦启动 + 可视化推进”工作流的人。
